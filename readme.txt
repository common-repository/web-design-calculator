== Description ==
<p>
This plugin is a tool to show Web Design Price or how much client would pay for web projects and so on. Please visit [demo](https://walexconcepts.com/wordpress/plugin-webestimator/).
</p>

Free Version features:
<p>
<ul>
<li>* Web Design Calculator looks responsive on any device.</li>
<li>* No backend or admin functionality .</li>
<li>* You cannot adjust the features according to your budget..</li>
<li>* Coding skills NOT required.</li>
<li>* Theme modifications NOT required.</li>
<li>* No animation circular progress bar.</li>
<li>* You cannot go beyond 3 steps to save and finish.</li>
<li>* You cannot change total currency symbol.</li>

</ul>
</p>
All

Premium Features:
<p>
<ul>
<li>* Web Design Calculator looks responsive on any device.</li>
<li>* Add backend or admin functionality .</li>
<li>* You can adjust the features according to your budget..</li>
<li>* Coding skills NOT required.</li>
<li>* Theme modifications NOT required.</li>
<li>* Add animation circular progress bar.</li>
<li>* You can go beyond 3 steps to save and finish.</li>
<li>* You can change total currency symbol.</li>
</ul>
</p>




== Installation ==
<p>
<ul>
<li>1. Upload files of webestimator `/wp-content/plugins/`</li> 
<li>2. Activate the plugin through the 'Plugins' menu in WordPress</li>
<li>3. At backend or admin page, put ShortCode /*[webestimator-guide]*/ to add Web Design Calculator to the page. Please always remember 
to remove [/* */] from the Shortcode.</li>
<li>4. At frontend, put ShortCode /*if ( shortcode_exists( 'webestimator-guide' ) ) { echo do_shortcode('[webestimator-guide]');} */ to 
add Web Design Calculator to the page. Please always remember to remove [/* */] from the Shortcode.</li>
<li>5. Premium Features, click 'Web Design Calculator' menu in WordPress to open webestimator admin page.</li>

</ul>
</p>
